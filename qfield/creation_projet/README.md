# Créer et déposer un projet QField sur le serveur QFieldCloud

1. Ouvrir un projet QGis vierge

2. Ouvrir la liste des projets QFields du serveur

![](../qfield_doc_img/img/100000000000002F0000002B136603BD.png)

![](../qfield_doc_img/img/1000000000000194000001B2F9223796.jpg "une-image-contenant-texte-capture-décran-nombre-logicielle-contenu-généré-par-lia-peut-être-incorrect.")

3. Créer un nouveau projet

![](../qfield_doc_img/img/10000000000000300000002E8706B1ED.png)

![](../qfield_doc_img/img/100000000000018E000001AC4E682D2B.jpg "une-image-contenant-texte-capture-décran-logiciel-policele-contenu-généré-par-lia-peut-être-incorrect.")

4. Donner un nom au projet sans préciser d'emplacement.

![](../qfield_doc_img/img/1000000000000183000001A2BB5FE82B.jpg "une-image-contenant-texte-capture-décran-affichage-logicielle-contenu-généré-par-lia-peut-être-incorrect.")

5. QGis propose un emplacement par défaut dans le profil utilisateur

![](../qfield_doc_img/img/10000001000001E30000020A97CFFBB3.png)

6. Enregistrer le projet QGIS dans le dossier du projet QField qu'on vient de créer

![](../qfield_doc_img/img/100000000000008C00000066ED9E143B.png "une-image-contenant-texte-capture-décran-police-diagrammele-contenu-généré-par-lia-peut-être-incorrect.")

![](../qfield_doc_img/img/10000000000001A7000001E1EE4160E0.jpg "une-image-contenant-texte-capture-décran-logiciel-affichagele-contenu-généré-par-lia-peut-être-incorrect.")

7. Copier les données sources (gpkg files) dans le même dossier

![](../qfield_doc_img/img/10000000000001A90000011EB4C8775A.jpg "une-image-contenant-texte-capture-décran-logiciel-icône-dordinateurle-contenu-généré-par-lia-peut-être-incorrect.")

8. Ajouter les données sources au projet QGIS

![](../qfield_doc_img/img/10000000000001A2000000EFEC0B94A4.jpg "une-image-contenant-texte-diagramme-carte-capture-décranle-contenu-généré-par-lia-peut-être-incorrect.")

9. Ouvrir la liste des projets QFields du serveur

![](../qfield_doc_img/img/100000000000002C00000030509A1DE4.png)

10. Sélectionner le projet QField qu'on vient de créer

![](../qfield_doc_img/img/10000000000001C1000001E78ACC7B5F.jpg "une-image-contenant-texte-capture-décran-logiciel-nombrele-contenu-généré-par-lia-peut-être-incorrect.")

11. Lancer la synchronisation

12. Téléverser les fichiers ou modifications

![](../qfield_doc_img/img/10000000000001C2000001C3DF7B7C88.jpg "une-image-contenant-texte-capture-décran-logiciel-icône-dordinateurle-contenu-généré-par-lia-peut-être-incorrect.")

**Le projet est disponible sur le serveur**
